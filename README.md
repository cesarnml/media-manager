### Frontend Boilerplate

1. Install webpack and webpack-cli

   - `npm i webpack webpack-cli`

2. Install eslint locally

   - `npm i eslint`

3. Change `package.json` to:

   - add `"private": true`
   - delete `"main": "index.js"`
